## 🎮 Nexora — Custom Matchmaking & Rating Platform (MAIN)

🌐 **Сайт:** https://nexora-game.ru  

**Nexora** — мой основной pet-проект и долгосрочная разработка.  
Игровая платформа с собственным матчмейкингом, системой party, инвайтами и логикой принятия матча.

---

### Ключевые возможности:
- Matchmaking (`searching → found → accepted → started`)
- Party до 5 человек
- Инвайты и подтверждение участия
- Ready / Accept система
- Автостарт матча при готовности всех игроков
- State-driven frontend без перезагрузки
- Подготовка к интеграции игрового бота

### Стек:
- **Backend:** PHP 8+, Laravel, PostgreSQL
- **Frontend:** HTML, CSS, Vanilla JavaScript
- REST API, Service Layer, polling-логика

🔗 Репозиторий: **[Nexora](https://github.com/0WiLe0/Nexora)**
